Complementary colors

Complementary Colors are colors that are in the contrast of one another on the color wheel.When asked to code about them you are often given their hex code.To code a complementary color you name the color you would like then you add a { } braket,put background-color: and the hex code. 


       .blue {
         background-color: blue;
       }
       .yellow {
         background-color: yellow;
       }

Adjust the Color of Various Elements to Complementary Colors

In this lesson we use two colors that are complementary to each other. Example: This page will use a shade of teal (#09A7A1) as the dominant color, and it's orange (#FF790E) complementary color.These colors opposite of each other on the color wheel can make each other appear more vibrant when placed side-by-side. 


      header {
        background-color: #09A7A1;
        color: white;
        padding: 0.25em;
      }
      h2 {
        color: #09A7A1;
      }
      button {
        background-color: #FF790E;
      }

Tertiary colors/ Background textures

There are three types of the colors that we mix in to create our colors in CSS and HTML. As i would like to call it the "three main sections of the cololrs combination."

Primary colors Are colors that are the first of the basics that make up all of our other colors. For example: Red,blue and Yellow

Secondary colors The colors that were made from the Primary colors in short as its second to the primary. For example: Orange, Green, Violet.

Tertiary colors are the colors made from Primary and Secondary colors, which would be for example: blue-violet,red-violet and yellow-orange, with it being the last of the combination

Gradients

gradient functions add a ombre visual to any element you apply it to. This can range from making your background to container look like a warm sunset, or making fun designs and stripes using this nifty function

What you can do with gradients

make a soft gradient from one color to another 


           div{
        background-image: linear-gradient(to right, pink, blue );
          }

Make a harsher gradient that repeats 


         div{
      background-image: repeating-linear-gradient(to right, purple 20%, blue 40%, green 40% );
        }

Hue and tone of color using HSL elements

Hue is the color wheel, saturation is the color intensity, and lightness is the the amount of white or black is added to it. Using hsl() we can use values from 0 to 360. For example, if we want the color blue, the first value will be the hue, 240, the 2nd value would be the saturation,100% and the last value would be the lightness,50%. To fine more unique colors, you can search up hsl sliders and in order jt will be from hue to saturation to lightness.


     color: hsl(hue number, saturation number, lightness number);
